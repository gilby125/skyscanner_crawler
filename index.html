    Project maintained by drZuo Hosted on GitHub Pages — Theme by mattgraham

    Description

    This program crawls http://www.skyscanner.net and lists possible combined flights destinations in given month. This might be helpful (a first step) when you plan your holidays.

    Download

    There are no binaries. Python sources are available at the project page.

    Requirements

    • Selenium (2.35)
    • Python (2.6)

    Python libs:

    • argparse
    • networkx
    • selenium

    Exemplary execution

    $  python SkyCrawler.py --price-limit 100 --currency pln --departure-point pl --language en
